📊 Mobile Sales Data Dashboard (Power BI)

🚀 Project Overview

This project is an interactive Mobile Sales Dashboard built using Power BI to analyze sales performance across different cities, brands, and time periods.
It provides meaningful insights into sales trends, customer behavior, and transaction patterns.

🎯 Key Highlights

💰 Total Sales: 769M
📦 Total Quantity Sold: 19K
🧾 Total Transactions: 4K
📊 Average Sales Value: 40K

📌 Features

📍 City-wise Sales Analysis using map visualization
📅 Monthly Sales Trend to track performance over time
⭐ Customer Ratings Analysis
💳 Payment Method Distribution (UPI, Debit Card, Credit Card, Cash)
📱 Top Selling Mobile Models Comparison
🏷️ Brand-wise Sales & Transactions
📆 Day-wise Sales Insights

🧰 Tools & Technologies Used

Power BI – Data visualization & dashboard creation
Excel Dataset – Data source
DAX (Data Analysis Expressions) – Calculations & measures
